# Lessons

## Purpose

To teach the user how to read and write hanzi and vocabulary from HSK

## Definitions

A **lesson** contains information about one hanzi or one word
The **reading** of a hanzi or word is its corresponding pinyin and tone
The **meaning** of a hanzi or word is its corresponding English dictionary definition

## Requirements

- Up to ten lessons should be added per day
- Lessons should be randomly picked from the user's current HSK level
  - Vocabulary can only be picked once all the constituent hanzi have been learned
- Each lesson session should start by presenting all the info for each new item
- After presenting info for each new item, the user should be quizzed on the meaning and reading of each item in random order
  - Users should repeat incorrectly answered items until no more items are left
- A hanzi or word may have more than one meaning associated with it
- A hanzi or word may have more than one reading associated with it

### Backlog

- Provide guidance on pronunciation by:
  - Indicating where special rules apply.
    See:
    - <https://en.wikipedia.org/wiki/Tone_sandhi#Mandarin_Chinese>
    - <https://en.wikipedia.org/wiki/Standard_Chinese_phonology#Tone_sandhi>
  - Using IPA and linguistic descriptions of how to physically pronounce words.
    See:
    - <https://github.com/AnthonyDickson/LearnMandarin/blob/main/LearnMandarin/Pinyin.swift>
    - <https://en.wikipedia.org/wiki/Pinyin>
    - <https://en.wikipedia.org/wiki/Standard_Chinese_phonology>
- Animation, video or diagram to illustrate the correct stroke order for hanzi
  See [jisho.org](https://jisho.org/search/%E6%98%8E%20%23kanji) for an example
- Vocabulary lessons should include the stroke order for each of the constituent hanzi
- Each lesson should provide examples of how the hanzi or vocabulary is used including common usage patterns
- Include audio recordings for each lesson on a best effort basis
- Relative prevalence of each meaning or reading
- Drawing input for hanzi lessons, mostly targeted at mobile devices
