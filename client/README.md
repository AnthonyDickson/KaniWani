# client

This is the SPA (Single Page Application) for KaniWani.
To view the web app, first [build the client](#development) and then [start the server](../server/README.md).

## Development

```sh
make init  # Perform initial setup 
make watch # Watch files, test and then build
make build # Build the client
make test  # Run the tests
make clean # Delete build artifacts
```
